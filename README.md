## Installation

```bash
yarn install
cp .env.example .env
yarn dev
```

## Basic Usage

```bash
# Development
yarn dev

# Production build
yarn build && yarn start
```

## Core Features

- **Web3 Integration**: Connect with cryptocurrency wallets using ethers.js and Reown AppKit
- **Real-time Communication**: Socket.io integration for live streaming responses
- **Modern UI**: Tailwind CSS with Framer Motion animations
- **Responsive Design**: Fully responsive interface for all devices
- **Documentation**: Built-in docs with Fumadocs and MDX support
